rt

Real-time Monitoring

WebSocket-powered updates every 2 seconds. CPU, memory, uptime, and active/idle status for every session without refreshing.

ctx

Context Window Tracking

Progress bar showing token usage against the 200k window. Green, amber, and red thresholds so you know when sessions run hot.

ab

Agentboard Integration

Projects with .agentboard show a full kanban view in the detail panel. Backlog, todo, in-progress, and done columns.

git

Git Status at a Glance

Current branch, last commit, dirty file count, and remote URL for each project without switching terminals.

det

Expandable Detail Panel

Click any card for a resizable split-screen with tabs: overview, git, agentboard, package.json, and CLAUDE.md.

shr

Remote Sharing

Share individual sessions to a remote board via GUID. Optional PIN protection. Anyone with the link can view your flights.

Under the Hood

5 Steps to Full Visibility

1

Process Scan

Parses ps aux to find all running claude CLI processes.

2

Project Resolve

Uses lsof to map each process to its working directory.

3

Context Read

Extracts token usage from .claude session files.

4

Enrich

Fetches git, package, CLAUDE.md, and agentboard data.

5

Broadcast

WebSocket pushes data to all browsers every 2 seconds.

Start in 30 Seconds

Requires Bun v1.0+ and macOS.

git clone https://github.com/Kotrotsos/trafficcontrol.git
cd trafficcontrol
bun install
bun start
